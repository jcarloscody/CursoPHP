é um gerenciador de dependencias do php, ou seja, tem instrucoes indicando quais bibliotecas o meu projeto 
precisa para funcionar por exemplo.




INICIAR
-> pode criar um arquivo composer.json
-> linha de comando
    composer init           //para criar os arquivos
    composer install        //para instalar as dependencias do composer. so é usado na primeira vez
    composer update         //atualizar copos
